## Описание приложения

VK Learning Platform - интерактивная платформа для обучения и обмена знаниями о маркетинге и рекламе в VK. Платформа включает курсы, уроки и материалы по таргетированной рекламе, SMM-стратегиям и продвижению в социальных сетях.

## Интеграция VK SDK

Для работы приложения используется VK SDK (JavaScript SDK). Конфигурация находится в файле `vk-config.js`.

### Основные методы:

1. **VK_CONFIG.init()** - инициализация VK SDK
2. **VK_CONFIG.getUserInfo()** - получение информации о текущем пользователе
3. **VK_CONFIG.logEvent()** - логирование событий для аналитики

## Установка и запуск

1. Клонируйте репозиторий
2. Откройте файлы в браузере или разверните на веб-сервере
3. Приложение автоматически инициализируется при загрузке
